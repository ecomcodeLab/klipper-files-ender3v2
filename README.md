# Beispiel-Konfiguration für Ender 3 V2

Willkommen zu meiner Sammlung von Konfigurationsdateien für den Ender 3 V2! Diese Dateien können als Beispiel oder Unterstützung für andere Benutzer dienen, die ihre eigenen Drucker anpassen möchten.

## Inhalt

- `printer.cfg`: Die Hauptkonfigurationsdatei für den Drucker.
- `macros/`: Ein Verzeichnis mit verschiedenen Macro-Dateien, die spezifische Funktionen oder Abläufe automatisieren.

## Hinweise

- Diese Konfigurationsdateien sind speziell auf mein Setup und meine persönlichen Vorlieben angepasst. Es kann sein, dass sie nicht optimal für dein Setup funktionieren.
- Ich übernehme keine Gewährleistung für die Richtigkeit oder Funktionalität dieser Dateien. Bitte überprüfe und teste alles sorgfältig, bevor du Änderungen an deinem Drucker vornimmst.

## Verwendung

1. Lade die Dateien herunter und speichere sie in den entsprechenden Verzeichnissen deiner Druckersoftware.
2. Passe die Konfigurationen nach Bedarf an, um sie an dein spezifisches Setup anzupassen.
3. Teste die Konfiguration gründlich, um sicherzustellen, dass alles wie gewünscht funktioniert.
